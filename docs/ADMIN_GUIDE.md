# Admin Guide

## Login
- Open `Account/Login`
- Use admin/ceo credentials

## Product Management
- Open Products page
- Use `Add Product` to create new product
- Fill:
  - Name
  - Department
  - Category
  - Subcategory
  - Description
  - Price
  - Stock
  - Image URL

The department/category/subcategory combination is validated against allowed taxonomy.

## Order and Payment Management
- Open Orders page
- Review customer details, payment method, reference, proof URL
- Update:
  - Order status
  - Payment status

## Role Permissions
- Employee: view orders/dashboard
- Admin: manage products/orders/reviews
- CEO: full access including delete actions
